Meu Site de Biblioteca Virtual

Este site foi criado com HTML e CSS para oferecer uma plataforma onde os leitores possam explorar e compartilhar livros. O objetivo é promover a leitura, apresentando informações sobre obras populares, incluindo autores, sinopses e links para aquisição.

A estrutura do site inclui um cabeçalho com o título principal e uma divisão principal (.container) que organiza a coleção de livros em uma grade (.livros). Cada livro é apresentado com uma imagem da capa, descrição breve e informações como autor e ano de publicação. O design responsivo assegura uma navegação fluida, adaptando-se a diferentes tamanhos de tela.

O CSS estiliza a página com cores suaves, uma tipografia acolhedora e botões interativos que incentivam a interação com o conteúdo. A simplicidade do HTML e CSS proporciona uma experiência visual agradável, sem necessidade de scripts avançados, promovendo a leitura de forma acessível e atrativa.

Página Inicial da TIM

O projeto é uma página web criada com HTML e CSS, desenvolvida para exibir os serviços e planos oferecidos pela TIM. O objetivo é criar um ambiente virtual que facilite a escolha do plano ideal, destacando promoções, detalhes dos serviços e benefícios exclusivos.

A estrutura do site utiliza HTML para organizar o conteúdo, com um cabeçalho exibindo o logotipo e o menu principal. A divisão principal (.container) apresenta as categorias de planos em seções flexíveis (.planos), destacando os detalhes de cada opção, preços e um botão para assinar. O layout responsivo permite que a página seja navegada com facilidade em diferentes dispositivos.

O CSS é responsável por estilizar a página, definindo um esquema de cores moderno com o azul característico da TIM, tipografia elegante e botões interativos que realçam a experiência do usuário. O design limpo e funcional garante que o site seja acessível e intuitivo, ajudando as pessoas a encontrar informações com rapidez e clareza.

O foco no design interativo e educativo transforma o aprendizado em uma experiência divertida e acessível. A escolha de cores vibrantes, textos claros e elementos animados torna o projeto envolvente, incentivando os participantes a explorar e aprender mais sobre ciência

Flash Card: Ferramenta Interativa de Estudo

Este projeto é um site interativo de perguntas e respostas desenvolvido no Scratch, criado para fornecer uma experiência de aprendizado prática e divertida. O site apresenta cartões com perguntas e respostas, permitindo que os usuários testem seus conhecimentos de forma dinâmica e envolvente.

A estrutura é simples e acessível, com cartões organizados para que cada pergunta possa ser revelada ao clicar, mostrando a resposta correspondente. O design do Scratch utiliza animações e botões interativos, facilitando a navegação e tornando o aprendizado intuitivo e acessível a todas as idades.

Com o Scratch, o projeto aproveita blocos de programação para controlar a interação entre os cartões e os usuários, criando uma plataforma educativa que torna o aprendizado mais atraente e acessível
